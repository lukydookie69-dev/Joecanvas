Get started

Install in 60 seconds

1

Download the extension

Click the download button and save the ZIP file anywhere on your computer.

2

Unzip the file

Right-click the ZIP → Extract All. You'll get a folder called canvas-drive-extension.

3

Open Chrome Extensions

Go to chrome://extensions in your browser and turn on Developer mode (top right toggle).

4

Load the extension

Click Load unpacked and select the unzipped folder. StudyLauncher will appear in your extensions.

5

Press Ctrl+Shift+Space

On any tab, press the shortcut to open the command bar and start building your workspaces.
